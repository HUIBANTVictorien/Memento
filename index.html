Petit memento


Ligne de commande


Pour créer un dossier :

mkdir "nom du dossier"

mkdir "nom du dossier" "nom du dossier" "nom du dossier" (créer plusieurs dossiers dans la même racine)

mkdir -p (parents, créer le chemin)


Pour créer un fichier :

touch "nom du fichier.extention"


Pour lancer une application :

"nom de l'application" lance l'application nommée (firefox/atom/slack et...)

atom index.html (lance index.html dans atom (possibilité d'ouvrir plusieurs fichiers en même temps)il faut se trouver dans le dossier)

firefox "nom du fichier html" ouvre le fichier dans un nouvel onglet


Pour afficher une liste des dossiers des fichiers et les manuels :

ls (affiche les dossiers et fichiers du dossier courant)

ls -a (affiche les dossiers/fichiers cachés)

ls -l (affiche les dossiers/fichiers en liste longue)

ls -al (affiche les dossiers/fichiers(même cachés) en liste longue)


Pour afficher le chemin du dossier :

pwd


Pour naviguer dans les dossiers :

cd "nom du dossier"

cd~ permet de revenir au dossier user(victorien)

cd .. permet de revenir au dossier précédent


Pour passer en mode super user :

sudo -s


Pour gérer les paquets :

apt update (met à jour les paquets)

apt upgrade (met à jour l'OS)


Poour rerchercher des fichiers :

apt search "nom recherché"

apt search "nom recherché" | (pipe) grep "mot recherché"
