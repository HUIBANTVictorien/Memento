La ligne de commande

Gestion de documents

Pour créer un dossier je dois utiliser cette commande : 


         mkdir "nom du dossier"

Pour afficher une liste des dossiers des fichiers et les manuels j'ai le choix de :


    
    
    ls (affiche les dossiers et fichiers du dossier courant)
    
    ls -a (affiche les dossiers/fichiers cachés)
    
    ls -l (affiche les dossiers/fichiers en liste longue)
    
    ls -al (affiche les dossiers/fichiers(même cachés) en liste longue)

Pour afficher le chemin du dossier j'utilise la commande : 

    
      pwd

Pour naviguer dans les dossiers j'utilise :

    
      cd "nom du dossier"
    
      cd~ (permet de revenir au dossier user(victorien))

Pour passer en mode super user : 

    
      sudo -s

Pour gérer les paquets

    
      apt update (met à jour les paquets)
    
      apt upgrade (met à jour l'OS)
